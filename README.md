# Подводная база — кликер

2D idle-кликер на C# (WPF) с **настоящими спрайтами** (подлодка, водоросли, реактор, комнаты базы). Ассеты — см. `Assets/CREDITS.md`.

## Запуск

```bash
cd UnderwaterBaseClicker
dotnet run
```

## Как играть

- **ТАП** по светящемуся реактору в центре — собираешь энергию.
- **Магазин справа** — покупай модули, они появляются на базе.
- Прогресс сохраняется при выходе.
